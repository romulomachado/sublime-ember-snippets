# test-helpers

## Usage

### andThen⇥

```js
andThen(() => {
  ${1}
});
```

### click⇥

```js
click('${1:selector}');
```

### fillIn⇥

```js
fillIn('${1:selector}', '${2:text}');
```

### find⇥

```js
find('${1:selector}', '${2:context}');
```

### keyEvent⇥

```js
keyEvent('${1:selector}', '${2:type}', '${3:keyCode}');
```

### pause⇥

```js
pauseTest();
```

### trigger⇥

```js
triggerEvent('${1:selector}', '${2:type}', '${3:options}');
```

### visit⇥

```js
visit('${1:url}');
```

### wait⇥

```js
wait();
```
